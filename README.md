 NDEJJE CANTEEN GRAB APP
Project Overview
The Ndejje Canteen Grab App is an application developed to improve the food ordering experience for students and staff at Ndejje University. The application provides a digital platform where users can easily browse available meals, place orders and reduce the need to physically queue at campus canteens.
By the aid of mobile technology, the system aims to enhance efficiency, save time and create a more organized food service environment within the university.
 Problem Statement
At the University, getting a meals from campus canteens has challenges that affect both students and staff. During peak hours, long queues often form leading to delays and time wastage while the congestion around the canteen area creates discomfort and inefficiency. There is limited knowledge of the meals available each day hence forcing users to physically visit the canteen before making decisions. It further slows down service delivery at the University. These issues negatively impact users’ time management and makes the overall food purchasing experience stressful and less efficient.

Installation Guide
Obtain the APK File
    Download the app-debug.apk file from;
                  A shared link ( WhatsApp, email etc.)  
Enable Installation from Unknown Sources
Since the app is not from the Play Store and then;
Go to Settings on your phone 
Navigate to Security or Privacy 
Enable Install Unknown Apps  
 Install the Application
Tap on the APK file 
Click Install 
Wait for the installation process to complete 
Tap Open after installation
